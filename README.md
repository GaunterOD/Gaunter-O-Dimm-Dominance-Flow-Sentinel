**GAUNTER-O-DIMM Price Action & Trend Spec v7.1**
과거 매물대(Support/Resistance)와 시장 점유율(Dominance) 데이터를 기반으로 한 가상화폐 기술적 분석 도구입니다.

---

## 🚀 v7.1 업데이트 (New Features)

### 🛡️ S/R Flip (지지/저항 전환) 감지
과거 500개 캔들 데이터를 분석하여 현재 가격이 주요 지지/저항 구간에 도달했는지 확인합니다.
- **Support Flip (🟢):** 과거의 고점(저항)이 현재의 지지 구간으로 작용하는 경우
- **Resistance Flip (🔴):** 과거의 저점(지지)이 현재의 저항 구간으로 작용하는 경우
- **가변 오차 범위:** 타임프레임에 따라 감지 범위를 자동 조절 (15분봉: 0.5% / 4시간봉: 1.0% / 일봉: 2.0%)

### 💵 정밀 USDT 모니터링
- 코인베이스(Coinbase) 데이터를 사용하여 테더 가격을 소수점 4자리($1.0001)까지 표시합니다.
- $1.00 페깅 범위를 벗어날 경우($0.9995 이하 / $1.0005 이상) 상태 메시지를 출력합니다.

## 📊 주요 기능 설명

### 1. 글로벌 마켓 뷰 (Global Market View)
비트코인과 테더의 점유율 변동을 통해 시장 자금 흐름을 파악합니다.

| 항목 | 설명 | 참고 사항 |
| :--- | :--- | :--- |
| **BTC.D** | 비트코인 도미넌스 | 상승 시 알트코인 약세 가능성 |
| **USDT.D** | 테더 도미넌스 | 상승 시 현금화 비중 증가 (시장 하락 가능성) |
| **USDT** | 테더 가격 ($) | 스테이블 코인의 페깅 상태 확인 |

### 2. 추세 지도 (Trend Map)
EMA 55일선과 200일선의 위치 관계를 분석합니다.
- 📈 **정배열:** 55선 > 200선 (상승 추세)
- 📉 **역배열:** 55선 < 200선 (하락 추세)

### 3. S/R 구간 분석
현재 가격이 과거 특정 시점의 고점이나 저점 부근에 도달했는지 알려줍니다.
- **출력 예시:** `🟢 지지(Support) 테스트 중 (기준: 2024-01-15 고점)`
- **의미:** 2024년 1월 15일의 고점 가격대에 현재 가격이 도달하여 지지 여부를 테스트 중임.
